## كيفية رفع المشروع على GitHub
لحفظ مشروعك على GitHub، اتبع الخطوات التالية من جهازك (Terminal):

1. أنشئ مستودعاً جديداً (New Repository) على [GitHub](https://github.com/new).
2. افتح Terminal في مجلد المشروع ونفذ الأوامر التالية بالترتيب:
   ```bash
   git init
   git add .
   git commit -m "إطلاق متجر THREAD"
   git branch -M main
   git remote add origin [رابط_المستودع_الخاص_بك]
   git push -u origin main
   ```

## إعدادات Firebase Console
1. **Enable Authentication**: فعل تسجيل الدخول بالبريد الإلكتروني وجوجل.
2. **Firestore Database**: أنشئ قاعدة بيانات في وضع الإنتاج.
3. **Storage**: فعل خدمة التخزين لرفع صور المنتجات.
